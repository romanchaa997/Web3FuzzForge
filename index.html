Interactive Demo

Generate Security Tests

Terminal 
# Generate a security fuzzing test
$ web3fuzzforge generate tx --wallet metamask \
  --out ./tests/security-tx-test.js --fuzz

Output

security-tx-test.js 
// Click "Generate Test" to see the output

Quick Start

Installation 
$ npm install web3fuzzforge
Basic Usage 
# Generate a wallet connection test
$ web3fuzzforge generate connect --wallet metamask --out ./tests/connection-test.js

# Run tests with a browser UI
$ web3fuzzforge run --headed
