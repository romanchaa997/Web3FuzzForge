# Web3FuzzForge Visual Assets

This folder contains the visual assets for the Web3FuzzForge project.

## Required Assets

Please place the following files in this directory:

1. `logo.png` - The Web3FuzzForge logo (40-50px height recommended)
2. `favicon.png` - Favicon for the website (32x32px recommended)
3. `web3fuzzforge-banner.png` - Main banner image for social sharing (1200x630px recommended)
4. `demo.gif` - Animated demo of the Web3FuzzForge tool in action

## Image Optimization

For best performance, please optimize all images before adding them:

- Use PNG format for logos and icons
- Use optimized JPG (80% quality) for photos
- Keep GIFs under 2MB if possible

You can use tools like TinyPNG (https://tinypng.com/) to optimize your images.
