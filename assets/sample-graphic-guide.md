# Web3FuzzForge Visual Assets Guide

This guide provides specifications and design suggestions for the visual assets needed for the Web3FuzzForge project.

## Banner Image (web3fuzzforge-banner.png)

**Specifications:**
- Dimensions: 1200 x 630 pixels
- Format: PNG with transparency
- Color scheme: Dark background (#19191F) with brand colors (#3778FF, #6147FF, #B947FF)

**Design Suggestion:**
- Create a dark tech-themed banner with a shield design containing blockchain network lines
- Include the text "Web3FuzzForge" prominently
- Add a tagline: "Security-first testing for Web3 applications"
- Consider adding a security lock symbol within the shield design

## Logo (logo.png)

**Specifications:**
- Dimensions: Square format, 500 x 500 pixels (will be displayed smaller)
- Format: SVG or PNG with transparency
- Style: Shield icon with blockchain network lines and security lock symbol

**Design Suggestion:**
- Use the primary blue (#3778FF) as the main color
- Include gradient elements with secondary purple (#6147FF) and accent pink (#B947FF)
- Clean, simple lines for good small-size rendering
- Should match the logo files in `docs-site/static/img/`

## Favicon (favicon.png)

**Specifications:**
- Dimensions: 32 x 32 pixels
- Format: PNG with transparency
- Style: Simplified version of the main logo
- Should match the `favicon.svg` in `docs-site/static/img/`

## Demo GIF (demo.gif)

**Specifications:**
- Dimensions: 800 x 450 pixels (16:9 ratio)
- Format: Animated GIF
- Max size: Try to keep under 2MB
- Duration: 6-10 seconds

**Content Suggestions:**
1. Show a terminal executing the Web3FuzzForge commands with CLI styling
2. Display test generation with branded colors
3. Show a test execution with MetaMask wallet popup
4. Show a security report output

## Color Palette

### Primary Colors
- Primary Blue: #3778FF (Main brand color, buttons, links)
- Secondary Purple: #6147FF (Accents, secondary buttons)
- Accent Pink: #B947FF (Highlights, gradient endpoints)
- Dark: #19191F (Dark mode backgrounds)
- Light: #F7F9FC (Light mode backgrounds)

### Semantic Colors
- Success: #2ECC71 (Success messages, confirmations)
- Warning: #F39C12 (Warnings, cautions)
- Error: #E74C3C (Error messages, critical alerts)
- Info: #3498DB (Information messages, tips)

## Typography

- Main Font: System UI stack (system-ui, -apple-system, Segoe UI, Roboto, etc.)
- Headings: 600 weight (semi-bold)
- Body: 400 weight (regular)

## UI Elements

- Border Radius: 8px
- Box Shadow: 0 4px 12px rgba(0, 0, 0, 0.1)
- Border Accents: 4px left border in brand colors

## Design Tools

Free tools that can help create these assets:
- [Canva](https://www.canva.com/) - Easy drag and drop design
- [Figma](https://www.figma.com/) - Professional design tool with free tier
- [GIMP](https://www.gimp.org/) - Free Photoshop alternative
- [ScreenToGif](https://www.screentogif.com/) - For creating demo GIFs
- [TinyPNG](https://tinypng.com/) - For optimizing images

## Brand Voice

When creating visual elements, keep the following brand voice in mind:
- Professional but approachable
- Technical but clear
- Security-focused but encouraging
- Confident but not arrogant
