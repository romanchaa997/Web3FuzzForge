# Web3FuzzForge Image Assets

This directory contains all image assets for the Web3FuzzForge project.

## Required Images

### Core Images
- `web3fuzzforge-banner.png` - The main banner image used in the README and landing page
- `web3fuzzforge-logo.png` - The project logo in PNG format
- `web3fuzzforge-logo.svg` - The project logo in SVG format
- `demo.gif` - An animated GIF showing the tool in action

### Vulnerability Guides
The `vulnerability-guides/` directory should contain images related to specific vulnerability types.

## How to Generate Images

### Banner Image
1. Open `temp/banner.html` in a browser
2. Adjust window size to match the banner dimensions (1200x400px)
3. Take a screenshot
4. Save as `web3fuzzforge-banner.png`

### Demo GIF
1. Open `temp/demo-gif.html` in a browser
2. Use the buttons to navigate through the steps
3. Take screenshots of each step
4. Combine screenshots into an animated GIF using a tool like:
   - [ezgif.com](https://ezgif.com/maker)
   - [GIMP](https://www.gimp.org/)
   - [Adobe Photoshop](https://www.adobe.com/products/photoshop.html)
5. Save as `demo.gif`

## Image Guidelines

- Keep file sizes optimized (use compression when possible)
- Maintain a consistent color scheme
- Use the following brand colors:
  - Primary Blue: `#60A5FA`
  - Dark Background: `#1E293B`
  - Light Text: `#CBD5E1`
  - Accent: `#F59E0B`

## Adding New Images

When adding new images:
1. Place them in the appropriate directory
2. Update this README if necessary
3. Ensure they are referenced correctly in documentation
