# VS Code Extension Images

This directory contains images used in the VS Code extension README and marketplace listing.

The following images should be created:

- **extension-banner.png**: Main banner image for the extension (recommended size: 1280×640)
- **one-click-generation.gif**: Animation showing the one-click test generation workflow
- **code-snippets.gif**: Animation demonstrating the code snippets in action
- **inline-documentation.png**: Screenshot showing the hover documentation feature
- **security-checks.png**: Screenshot of the security checks feature

## Icon Guidelines

- **Web3FuzzForge Icon**: Use the shield logo with the blue-purple gradient background
- **Feature Icons**: Use VS Code style icons with consistent coloring
- **Screenshots**: Use the dark theme for all screenshots

## GIF Recording Guidelines

When recording GIFs:
1. Use a clean VS Code window with the dark theme
2. Demonstrate the feature clearly and concisely
3. Keep the recording under 10 seconds
4. Optimize the GIF for web display (reduce file size)

## Image Formats and Sizes

- PNG images should be optimized for web
- GIFs should be kept under 2 MB
- All images should have appropriate dimensions for the VS Code Marketplace
